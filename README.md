Weather App
-----------

* Created and wrote code for an app that allows users to look up the weather in different cities in real time, and also provides the possibility to access further weather details by clicking the city name. 
* Designed, developed, and tested site using HTML, CSS3, and React JS technologies, meeting accessibility and web browser standards for websites.
